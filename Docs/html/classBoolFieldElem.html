Подробное описание

Класс, реализующий элемент поля из 2 ^ k элементов

Реализует работу с поляризованным полиномом над полем из

\[2 ^ {BoolFieldElem::irreducible\_polynominal.size()} \]

элементов
Перед первой инициализацией переменной класса BoolFieldElem должен быть явно
указан BoolVec::irreducible_polynominal
irreducible_polynominal - неприводимый полином необходимой степени. Пример получения такого

полинома есть в примере использования библиотеки FieldVec Example1

Лучше использовать этот класс, чем FieldElem при FieldVec::q == 2.
