Подробное описание

Класс, реализующий элемент произвольного поля.

Реализует работу с поляризованным полиномом над полем из

\[FieldVec::q ^ {FieldElem::irreducible\_polynominal.size()} \]

элементов
Перед первой инициализацией переменной класса FieldElem должны быть явно
указаны FieldVec::q и FieldElem::irreducible_polynominal
q должно принимать значения >= 2
irreducible_polynominal - неприводимый полином необходимой степени. Пример получения такого

полинома есть в примере использования библиотеки FieldVec Example1
