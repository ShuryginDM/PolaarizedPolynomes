Подробное описание

Класс, реализующий поляризованный моном над полем четной характеристики

Реализует работу с поляризованным полиномом над полем из

\[2 ^ {FieldElem::irreducible\_polynomial.size()} \]

элементов
Перед первой инициализацией переменной класса v должны быть явно
указаны BoolFieldElem::irreducible_polynomial и BoolPolarizedMonomial::polarization

Предупреждения
одновременно возможна работа только с мономами с одинаковой поляризацией

Лучше использовать BoolPolarizedMonomial, чем PolarizedMonomial с FieldVec::q == 2
